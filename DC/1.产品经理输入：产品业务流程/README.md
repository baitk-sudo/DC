# 1. 产品经理输入：产品业务流程（原始数据）

本目录存放**产品经理侧**提供/维护的产品业务流程原始数据（以 CSV 为主），用于后续的：

- 产品流程步骤级组件映射与满足性分析（见 `4.高阶设计-产品流程/`）
- 规则/配置落点讨论与回归检查（见 `2.高阶设计-横切设计/`）

## 使用提示

- CSV 建议以 **UTF-8** 打开（Excel 如出现乱码，请在导入时选择 UTF-8 编码）。
- 若你在本目录做了“规则调整补充”，建议同时更新横切设计中的：
  - `2.高阶设计-横切设计/组件设计-风控组件与规则触发总览.md`
  - `2.高阶设计-横切设计/产品规则-回归用例矩阵.md`
